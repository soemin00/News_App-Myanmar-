# 🇲🇲 NewsMM – Real-Time News App for Myanmar

**NewsMM** is a real-time news application built using **Flutter**, **Firebase**, and **NewsAPI**, aimed at providing Myanmar citizens with fast, reliable, and localized news. It empowers users to stay informed with updates from trusted sources and also allows community members to contribute real-time local news, helping close the gap between mainstream and grassroots journalism.

---

## 📌 Project Overview

In Myanmar, many people rely on platforms like Facebook for local news. However, vital information often gets lost in the noise. **NewsMM** was created to solve this by offering:

- A **dedicated news platform** free from distractions
- **Reliable news** from trusted sources like BBC Burmese, VOA, and RFA
- **User-submitted reports** to boost community participation
- **Location-based filtering** to help users stay informed about local events

Developed as part of our final-year major project at Marwadi University, this app blends journalism with civic tech.

---

## 🚀 Features

- ✅ Real-time news from trusted media outlets
- 🗣️ Community-powered local news posts
- 📍 Location-based news filtering
- 🔐 Firebase Authentication (Login/Signup)
- 👤 Profile and post management
- 📸 News image uploads via Cloudinary
- 📱 Clean, intuitive UI

## 🛠️ Installation Guide

1. **Clone the Repository**

```bash
git clone https://github.com/soemin00/News_App-Myanmar-.git
cd News_App-Myanmar-

2. **Install Dependencies**

flutter pub get

3. **Configure APIs**
  * Add your Firebase configuration in firebase_options.dart
  * Set your NewsAPI key in your API service
  * Set your Cloudinary credentials if using image upload

4. **Run The App**

flutter run
